# SF Football Club Raffle Backend

Express.js + MongoDB + Redis + Socket.io API

## Quick Start

```bash
cp .env.example .env   # or use pre-filled .env
npm install
npm run seed
npm run dev
```

API: http://localhost:5000/api/health
